I Convert a m4a/mp3 to 8bit WAV using ffmpeg recently!

Steps:

1. Download and Install "ffmpeg" windows build,

2. run `D:\soft\ffmpeg> .\bin\ffmpeg.exe -i .\luocha2.mp3 -ar 8000 -ac 1 -acodec pcm_u8 output3.wav`
